Speaker Recognition using Vector Quantization
------------------------------------------------------------------------------------
1. Copy-paste the folder 'Speaker Recognition System using Vector Quantization' anywhere on the computer.

2. Start MATLAB

3. Browse to the folder copy-pasted in Step 1.

4. Open file 'recognition.m'.

5. Run the selected file.
